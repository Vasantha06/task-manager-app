# 🚀 Smart Task Manager

A full-stack Task Manager web application where users can register, login, and manage their tasks efficiently.

---

## 🔧 Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap

---

## ✨ Features
- 🔐 User Authentication (Signup/Login)
- 🧾 Create, Read, Update, Delete Tasks (CRUD)
- 🔍 Search Tasks
- 📊 Filter Tasks (Pending / Completed)
- ⚠️ Due Date Highlight (Overdue in Red)
- 🎨 Clean UI with consistent theme

---

## 📁 Project Setup
```bash
1. Clone Repository
git clone https://github.com/Vasantha-06/task-manager-app.git
cd task-manager-app
2. Install Dependencies
npm install
3. Run Server
node app.js
4. Open in Browser
http://localhost:3000
```
⚠️ Note
Make sure MongoDB is running locally
Uses port 3000
